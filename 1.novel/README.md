# Python Novel Crawler

一个用于爬取小说章节的Python脚本。

## 项目简介

本子项目是一个简单的网络爬虫，用于从指定网站爬取小说章节内容。使用requests库发送HTTP请求，lxml库解析HTML内容。


## 依赖项

- Python 3.11
- requests
- lxml


## 注意事项

- 请确保遵守目标网站的robots.txt规则
- 建议设置合理的请求间隔，避免对目标网站造成过大压力
- 代理设置需要根据实际情况配置
