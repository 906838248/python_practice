# Python Lottery System

一个基于Flask的Web抽奖系统。

## 项目简介

本子项目是一个简单而美观的Web抽奖系统，使用Flask框架构建。用户可以通过网页界面输入自定义的抽奖名单，系统会随机抽取一名中奖者。项目包含基础版本和进阶版本两个实现，适合学习Flask框架和Web开发。

## 功能特性

### 基础版本 (lottery.py)
- 简单的Web抽奖功能
- 预设的抽奖名单
- 随机抽取中奖者
- 基础的HTML模板展示

### 进阶版本 (lottery_pro.py)
- 支持用户自定义输入抽奖名单
- 美观的渐变色UI设计
- 响应式布局，支持移动端
- 名单以标签形式展示
- 中奖结果带有动画效果
- 表单验证和数据处理
- 支持多行文本输入，自动去除空行

## 依赖项

- Python 3.x
- Flask 2.x
- 无需额外安装，Flask自带模板引擎

## 安装

1. 确保已安装Python 3.x
2. 安装Flask：
```bash
pip install flask
```

## 技术要点

- **Flask路由**：使用`@app.route`装饰器定义URL路由
- **模板渲染**：使用`render_template`渲染HTML模板
- **表单处理**：使用`request.form`获取POST数据
- **URL参数**：使用`request.args`获取GET参数
- **随机选择**：使用`random.choice`从列表中随机选择
- **CSS样式**：使用渐变色、动画效果美化界面
- **响应式设计**：使用Flexbox布局，支持移动端

## 注意事项

- 本项目仅在本地运行，未配置生产环境
- 抽奖名单存储在内存中，刷新页面后会重置
- 建议在开发环境中使用`debug=True`，生产环境请关闭
- 如需持久化存储，可以考虑使用数据库或文件
