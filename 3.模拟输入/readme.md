# 模拟输入

本项目起因是因为学习通部分课程的作业禁止粘贴，虽然可以通过网页开发工具粘贴上去但是太麻烦了，故而自己写了一个模拟输入软件。

## 功能说明

通过模拟键盘按键，将文本内容或剪切板内容输入到当前焦点窗口中。

- **文本输入**：在软件文本框中输入内容，按快捷键将内容逐字输入到目标窗口
- **剪切板输入**：将内容复制到系统剪切板，按快捷键将剪切板内容输入到目标窗口

## 快捷键

| 功能 | 快捷键 |
|------|--------|
| 输入文本框内容 | `F8` |
| 输入剪切板内容 | `F9` |

## 使用方法

1. 运行 `main.py`
2. 方式一：文本输入
   - 在软件上方的文本框中输入要输入的文字
   - 切换到目标窗口（如学习通作业输入框）
   - 按 `F8` 键，文本内容将被逐字输入
3. 方式二：剪切板输入
   - 将需要的内容复制到系统剪切板（`Ctrl+C`）
   - 切换到目标窗口
   - 按 `F9` 键，剪切板内容将被输入

## 依赖环境

- Python 3.9+
- tkinter（Python 内置）
- pyautogui
- keyboard

安装依赖：
```bash
pip install pyautogui keyboard
```

## 注意事项

- 运行程序需要管理员权限（`keyboard` 库需要监听全局键盘事件）
- 输入时目标窗口需要处于焦点状态，否则内容会输入到其他窗口
