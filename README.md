# Python Practice

本项目是一个Python脚本集合，用于练习和学习Python编程。

## 项目简介

本项目包含多个Python练习脚本，涵盖了网络爬虫、数据处理等常见的Python应用场景。每个子目录代表一个独立的练习项目，旨在通过实际编码练习来提升Python编程能力。



## 包含项目
### 1.爬虫相关
- **1.novel**：一个基于requests和lxml库的网络爬虫，用于从小说网站抓取小说内容。

### 2.web相关
- **2.lottery**：一个基于Flask的抽奖系统，用于随机抽取名单中的一个人作为中奖者。
